# 🎬 AI Movie Review Sentiment Analyzer

An end-to-end NLP project that performs sentiment analysis on IMDB movie reviews using DistilBERT.

## 🚀 Features

- Sentiment Prediction
- Confidence Score
- Example Reviews
- Prediction History
- Analytics Dashboard
- Streamlit Web App

## 📊 Model Performance

| Model | Accuracy |
|---------|---------|
| TF-IDF + Logistic Regression | 89.21% |
| DistilBERT | 91.46% |

## 🧠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Hugging Face Transformers
- DistilBERT
- PyTorch
- Streamlit
